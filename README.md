# Java Repository – Study

Este repositório contém exemplos e exercícios de estudo sobre a linguagem Java.

 Objetivo:
- Consolidar fundamentos da linguagem
- Servir como material de revisão
- Organizar conceitos por tema

📂 Organização:
- basics = sintaxe básica, entrada de dados, bitwise
- controlflow = condicionais e laços
- oop = classes, objetos e encapsulamento
- collections = List, ArrayList, for-each
- functional = Predicate, Stream, Comparator
- generics = Generics e wildcards
- datetime = LocalDate, Instant, Formatter
- io = Leitura e escrita de arquivos

 Projetos completos (JDBC, OO, Spring) estão em repositórios separados.
